# MondoMago — Prompt Suno AI per le 9 canzoni

## Come usare Suno

1. Vai su **https://suno.com** → accedi con Google
2. Clicca **"Create"** → scegli **"Custom"** (non "Instrumental")
3. Incolla il blocco **[Style]** nel campo "Style of Music"
4. Incolla le **Lyrics** nel campo "Lyrics"
5. Titolo: usa il nome indicato sotto
6. Genera → seleziona la versione migliore → **Download MP3**
7. Rinomina il file come indicato e mettilo in `public/audio/`

**Nota:** Suno genera canzoni di circa 30-90 secondi. Genera 2 versioni per ogni mondo e scegli quella più orecchiabile.

## Dopo il download — Attivare nell'app

1. Metti i 9 file MP3 in `public/audio/` con i nomi indicati sopra
2. In `src/MondoMago.jsx`, cerca `WORLD_SONGS` e trova il mondo corrispondente
3. Cambia `fullSong: false` → `fullSong: true` per quel mondo
4. Fai il build e il deploy

**Esempio:**
```js
foresta: {
  title: "La Canzone della Foresta",
  fullSong: true,   // ← cambia questo
  ...
}
```

## Script di split automatico (opzionale)

Se Suno genera una canzone troppo lunga o vuoi dividere un MP3 lungo in parti, usa:

```bash
# Installa ffmpeg se non presente
sudo apt install ffmpeg

# Divide song_foresta.mp3 in 8 parti uguali (24 secondi ciascuna se la canzone dura ~3 min)
DURATION=$(ffprobe -v quiet -show_entries format=duration -of csv=p=0 public/audio/song_foresta.mp3)
SEGMENT=$(python3 -c "print($DURATION/8)")
ffmpeg -i public/audio/song_foresta.mp3 -f segment -segment_time $SEGMENT \
  -c copy public/audio/song_foresta_%d.mp3
# Rinomina: _0.mp3 → _0.mp3, _1.mp3 → _1.mp3, ecc.
```
